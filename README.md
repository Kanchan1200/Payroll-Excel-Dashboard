Payroll Excel Dashboard 📊
Project Overview

This project is an interactive Payroll Dashboard built in Microsoft Excel to analyze employee salary data and generate insights for payroll reporting.

The dashboard helps track total payroll expenses, deductions, and net salary distribution across departments.

Key Features

Payroll KPI Summary (Total Salary, Total Deductions, Net Pay)

Department-wise Salary Analysis

Employee Payroll Breakdown

Interactive Charts & Visualizations

Clean and Structured Data for Reporting

Tools & Skills Used

Microsoft Excel

Pivot Tables

Excel Formulas

Data Cleaning

Conditional Formatting

Data Visualization

Dashboard Insights

The dashboard provides:

Overview of payroll expenses

Department-wise salary comparison

Employee salary distribution

Quick KPI metrics for management reporting

Payroll Dataset

Use Case

This dashboard can be used by HR teams, MIS analysts, and finance departments to monitor payroll data and generate reports efficiently.
